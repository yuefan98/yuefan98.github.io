Open Source

nleis.py provides a documented workflow for nonlinear impedance analysis.

nleis.py is documented as an impedance.py toolbox for paired linear and nonlinear circuit elements, simultaneous EIS and 2nd-NLEIS analysis, data truncation, fitting, visualization, and measurement-model validation.

  1. Set up the environment and install nleis.py from PyPI.
  2. Import frequency, Z1, and Z2 data from the experiment.
  3. Process the data with the documented nleis.py data-processing workflow.
  4. Define paired EIS and NLEIS circuit models using supported element pairs.
  5. Fit, visualize, and inspect results through the package APIs.
